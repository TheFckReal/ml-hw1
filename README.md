# Общая структура домашнего задания №1 по машинному обучению ИИ ВШЭ

1. `AI_HW1_Regression_with_inference_pro_pt1.ipynb` - ноутбук с домашним заданием, частью 1. `REPORT.md` - markdown файл с отчетом, требующимся для ноутбука.
2. `models` - артефакты (pickle файлы) обученных моделей/трансформеров. Оттуда же берутся веса для приложения.
3. `app.py` - главный файл streamlit-приложения.

## Структура приложения

1. `main` - функция запуска приложения, производит загрузку данных (модели, pipeline).
2. `EDA/show_eda` - страница разведочного анализа данных. Выводит информацию о распределении таргет переменной, графики зависимости численных переменных, а также общую корреляцию между фичами (через `phik`) после базовой предобработки признаков.
3. `Prediction/show_prediction` - страница предсказания по данным, позволяет ввести вручную наблюдение/загрузить `csv` файл и выгрузить предсказания.
4. `Model Weights/show_weights` - вывод весов моделей/важности признаков после всех предобработок на конечной модели. Позволяется выбрать `N` для `TOP N` фичей.

P.S. В оформлении, документировании, а также исправлении багов приложения помогали: Gemini 3 Pro (так себе) и Opus 4.5 (мега круто)
